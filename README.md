## 📖 About

Searching for cybersecurity interview resources is scattered and time-consuming. This repo solves that by organizing **the best free resources** into a single, structured guide — so you can focus on studying, not searching.

```
└──────────────────────────────────────────────────────────┘
```

### 🔹 Scenario-Based Questions — How to Answer

```
1. Understand the situation  →  What is happening?
2. Identify the threat       →  What type of attack / issue?
3. Contain & respond         →  What's the first action?
4. Escalate if needed        →  Who do you notify?
5. Document & learn          →  Post-incident review
```

## 📚 Study Strategy

A recommended **3-phase approach** to maximize your prep:

```
Phase 1 ─ Foundation (Week 1-2)
├── Start with TryHackMe & LetsDefend
├── Focus on SOC-style thinking
└── Practice alert triage scenarios

Phase 2 ─ Theory (Week 3)
├── Use Indeed & GUVI for broader theory
├── Cover encryption, networking, access control
└── Review common Q&A formats

Phase 3 ─ Polish (Week 4)
├── Use InfosecTrain & Test-King
├── Sharpen interview phrasing
└── Practice scenario handling out loud
```

## 📋 Key Topics Cheat Sheet

Quick-reference table of **must-know topics** by category:

| Category | Topics |
|---|---|
| **Networking** | OSI Model, TCP/IP, DNS, DHCP, ARP, HTTP/S, Ports, Subnetting, VPN |
| **Security Tools** | SIEM (Splunk, QRadar, Sentinel), IDS/IPS (Snort, Suricata), EDR, Firewalls, WAF |
| **Threats & Attacks** | Phishing, Ransomware, DDoS, MITM, SQL Injection, XSS, CSRF, Brute Force |
| **Incident Response** | IR Lifecycle (NIST), Containment, Eradication, Recovery, Lessons Learned |
| **Frameworks** | MITRE ATT&CK, NIST CSF, ISO 27001, CIS Controls, Kill Chain, Diamond Model |
| **OS & Logs** | Windows Event Logs, Linux Syslog, Registry, Processes, Services, Permissions |
| **Malware** | Types (Trojan, Worm, RAT, Rootkit), Static vs Dynamic Analysis, Sandboxing |
| **Cryptography** | Symmetric vs Asymmetric, Hashing (MD5, SHA), TLS/SSL, PKI, Digital Signatures |
